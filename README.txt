80 лет Калининградской области
Интерактивное веб-приложение, посвящённое 80-летию Калининградской области (1946–2026). Проект позволяет исследовать ключевые исторические, научные и культурные события региона через динамическую карту и хронологический список.

Возможности
Интерактивная карта с маркерами событий и подписями под точками.
Фильтр по диапазону лет – отображаются только события выбранного периода (слайдер).
Детальный список событийсправа с описанием, научными фактами и источниками.
Всплывающие подсказки при наведении на маркер.

Технологии
Python – основной язык.
Streamlit– фреймворк для создания интерактивных дашбордов.
Pydeck – библиотека для геовизуализации (слой точек и текстовых подписей).
Pandas – обработка и фильтрация данных.
